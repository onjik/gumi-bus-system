# gumi-bus-system

## 프로젝트 개요
구미시의 시내 버스 시스템은 시간, 경로 예측이 제대로 동작하지 않는다.

실시간 예측 시스템으로 사용자에게 정확한 대중교통 환승 정보를 제공하기 위해 시작했다.

## 제공 기능
1. 시내버스 도착 시간 예측
2. 대중교통 환승 정보

## 구성된 서비스 

### 도착 정보 서비스
- 버스의 정류장 위치 + 시간 조합을 제공

### 노선 내 소요시간 계산 서비스
- 노선 내 두 정거장 간 소요시간을 계산하는 API 제공

### 노선간 네비게이션 서비스
- 위치간 환승 경로를 계산하는 API 제공
